# GO-jwt-auth
JWT API Authentication using:
 GO
 JWT
 MongoDB

### Endpoints : 
```
POST http://localhost:8000/register
```

```
POST http://localhost:8000/login
```

```
GET http://localhost:8000/refresh
```

#### Install
```sh
$ git clone https://github.com/FocusCript/go-jwt-mongo
$ cd golang-mongodb-jwt
$ go get
```
#### Building 
```sh
$ go build
```
